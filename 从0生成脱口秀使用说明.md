# 从 0 生成一篇脱口秀使用说明

## 1. 打开空白创作态

进入 RiffMaster 页面后，如果没有真实草稿数据，编辑区会显示“从空白开始创作”。点击顶部工具栏或空白卡片里的“从0生成”。

## 2. 填写生成信息

建议按这个顺序填写：

- 主题 / 经历：写真实、具体的生活经历，例如“第一次租房被中介绕晕”“离职后假装很自由”“健身三天就买装备”。
- 你的态度：写你对这件事的真实情绪，例如“嘴硬但心虚”“表面成熟其实很慌”“明明受伤还要装洒脱”。
- 风格偏好：写喜剧手法，例如“观察喜剧 + 自嘲 + Callback”“更犀利一点”“温和但有反转”。
- 观众对象：写谁会听，例如“一二线城市打工人”“大学生”“亲密关系观众”。
- 目标时长：第一次建议 5-8 分钟。
- 限制 / 禁忌：写不要碰的方向，例如“不要低俗”“不要攻击具体群体”“不要地域黑”。

## 3. 生成初稿

点击“生成初稿”。系统会调用 `/api/ai/show`，让 AI 按 Hook、Premise、Setup、Punchline、Tag、Callback 生成一组可编辑段落。生成成功后，稿件会自动保存到 `/api/shows/prototype` 对应的真实草稿数据里。

## 4. 修改和追问

初稿生成后，建议按这个流程继续打磨：

1. 先通读一遍，删掉不像你说话的段落。
2. 点“AI分析”，看整体结构、节奏和弱段提示。
3. 在右侧问 Riff AI：“哪里最弱？”“帮我重写第 3 段”“给第 5 段加三个 Tag”。
4. 用“写作流程”逐步补 Premise、Angle、Setup、Punchline、Tag、Callback。
5. 拖拽段落调整顺序，让前 30 秒先有 Hook，中段不断升级，结尾回扣开头关键词。

## 5. 判断一篇初稿是否可继续排练

满足下面四点，就可以进入试讲：

- 开头 30 秒能让观众知道“你在讲什么”和“你为什么在意”。
- 每 1-2 段至少有一个明确 Punchline 或 Tag。
- 中段不是平铺经历，而是不断升级荒诞程度。
- 结尾能 Callback 开头关键词或核心处境。
